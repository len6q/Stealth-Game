# Stealth Game
### Стелс-игра с процедурно генерируемыми препятствиями
[_Видео геймплея_](https://youtu.be/oOjHTwpPu8o)

**В проекте реализованы:**

_1. Алгоритм обхода в ширину для создания игрового поля с как минимум одним проходом к точке финиша;_

_2. Factory Pattern для создания разных врагов;_

_3. Работа с NavMesh;_

_4. AI врага. Возможность патрулирования территорий, преследования игрока, потеря игрока из виду (реализовано при помощи State Pattern);_

_5. Создание field of view у врагов._
